# 协程启动方式

|   |   |
|---|---|
|DEFAULT|立即进入待调度状态|
|LAZY|只有需要（start/join/await）时才开始调度|
|ATOMIC|和DEFAULT类似,且在第一个挂起点前不能被取消|
|UNDISPATCHED|立即在当前线程执行协程体，直到遇到第一个挂起点（后面取决于调度器）|
